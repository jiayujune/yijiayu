# Jiayu Yi Portfolio

Personal portfolio website for GitHub Pages deployment.

## Overview

This repository contains the source for a static personal website that highlights:

- background and education
- selected projects
- experience
- contact links

## Tech Stack

- HTML
- CSS
- JavaScript

## Local Preview

Open `index.html` directly in a browser, or run a simple local server.

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

This site is intended to be deployed with GitHub Pages.
